## Install & Run
- Install requirements
```
npm instal
```

- Install mongo docker container
```bash
docker run -p 27017:27017 --name comedy-mongo -d mongo --storageEngine wiredTiger
```

- Run back server
```bash
npm run server
```

- Run front dev server
```
npm run webpack-devserver
```
